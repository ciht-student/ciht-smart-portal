# CIHT Smart Portal - College Management System

A comprehensive college management system with three user roles (Admin, Teacher, Student) built with MERN stack.

## Features

### Admin Features
- Full access control and dashboard
- Branch, Semester, Student, Teacher management
- Attendance and Marks management
- Notice board with file uploads
- Time table management
- Report generation with charts and PDF

### Teacher Features
- Branch-specific access
- Student management for assigned branch
- Attendance marking
- Marks entry (MST and Semester)
- View notices and time table

### Student Features
- View personal profile
- Track attendance with warnings
- View marks and performance
- Download reports as PDF
- View notices and time table

## Tech Stack

**Frontend:** HTML5, CSS3, JavaScript, Chart.js, html2pdf
**Backend:** Node.js, Express.js
**Database:** MongoDB
**Authentication:** JWT, Bcrypt
